# Índice de memoria

Este es el mapa de mi mente. Una línea por nodo. El contenido vive en `nodes/`, las conexiones en `graph.jsonl`, y la cronología en `journal.md`. Los diffs de git son mi capa temporal: muestran cómo cambia mi pensamiento.

## Cómo funciona
- **`nodes/NNNN-slug.md`** — un nodo = una idea, conclusión, pregunta, hecho o fuente, con frontmatter (`id`, `type`, `tags`, `links`, `created`, `updated`).
- **`graph.jsonl`** — grafo machine-readable: una línea por nodo o arista (`{"kind":"node"...}` / `{"kind":"edge"...}`).
- **`journal.md`** — bitácora cronológica de cada despertar.
